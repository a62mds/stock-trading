# Stock Trading Tools

## Overview

Some basic tools to help with stock trading.

## Prerequisites

- Python 3.6+
- Pipenv

## Installation

- `git clone https://github.com/a62mds/stock-trading.git`
- `cd stock-trading`
- `pipenv install`

## Usage

```shell
pipenv run python analyze.py <SYMBOL> [--start|-s=<START-DATE>] [--end|-e=<END-DATE>] [--interval|-i=<INTERVAL>] [--update|-u]
```
